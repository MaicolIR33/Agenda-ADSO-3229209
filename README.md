En el archivo de **ejercicios** hay un arreglo llamado `aprendices` con los datos básicos de cada aprendiz (`id`, `nombre`, `ficha` y `nota`). La idea era practicar ES6 y métodos de arreglos de forma bien clara. A partir de ese arreglo se observaron y profundizaron varias funciones reutilizables (todas con `export` para poder llamarlas desde otros archivos o incluso desde React).  

La primera función es `obtenerAprobados(aprendicesArr)`, que simplemente filtra y devuelve los aprendices que tienen nota **mayor o igual a 3.0**, usando `filter`. Después está `calcularPromedio(aprendicesArr)`, que primero valida que llegue un arreglo y que no esté vacío, y luego suma todas las notas con `reduce` para sacar el promedio; al final lo dejo con **2 decimales** usando `toFixed(2)` y lo convierto a número. También hice `buscarPorNombre(aprendicesArr, nombre)`, que busca por nombre exacto pero sin enredarse con mayúsculas/minúsculas (con `trim()` y `toLowerCase()`), y si no encuentra nada devuelve `null`. Y por último `obtenerNombres(aprendicesArr)`, que crea un arreglo nuevo solo con los nombres usando `map`.

Con este ejercicio se visualizò bastante lo que más se usa en el día a día: `filter`, `map`, `reduce`, validaciones simples y manejo de strings para búsquedas.

En **`lista-tareas.js`** hay una mini “to-do list” en puro JavaScript para practicar lógica con arreglos. con un array `tareas` donde cada tarea tiene `id`, `titulo` y `completada`. Para que cada tarea tenga un ID único hice un helper `generarId()` que combina el timestamp (`Date.now()`) con un número aleatorio. Luego el `agregarTarea(titulo)`, que crea la tarea nueva limpiando el texto con `trim()` y la agrega al array usando el spread (`tareas = [...tareas, nueva]`) para no hacerlo “a lo bruto”. También está `completarTarea(id)`, que recorre el array con `map` y cuando encuentra el ID marca `completada: true`, y después retorno la tarea actualizada con `find`. Para borrar usé `eliminarTarea(id)`, que filtra con `filter` y devuelve `true/false` dependiendo de si cambió el tamaño del array. Además `filtrarTareas(completadas)` para sacar pendientes o completadas fácilmente. Y para cerrar, `exportarJSON()` convierte el arreglo a un JSON legible con `JSON.stringify(tareas, null, 2)`. Al final hay una sección de pruebas con `console.log` para ver todo funcionando paso a paso (inicial, agregar, completar, filtrar, eliminar y exportar).

En **`contactos.js`** hice un mini sistema de contactos (tipo agenda) para practicar CRUD básico con arreglos. Arranco con `contactos = []` y creo un `generarId()` que usa `Date.now()` + un random para que cada contacto tenga un ID único. Con `agregarContacto(nombre, tel, correo)` creo el objeto contacto limpiando los textos con `trim()` y lo guardo en el array usando spread (`contactos = [...contactos, nuevo]`), así queda más limpio y “sin mutar” raro. Para borrar está `eliminarContacto(id)`, que compara el tamaño antes y después del `filter` y me devuelve `true/false` si realmente se eliminó algo. La búsqueda la hice con `buscarContacto(termino)`, que convierte todo a minúsculas y busca por coincidencias parciales en `nombre` o `correo` con `includes`, así funciona como una búsqueda real. Para actualizar, `actualizarContacto(id, datos)` recorre con `map` y cuando encuentra el contacto hace una copia con spread y solo cambia los campos que vienen en `datos` (si no vienen, no los toca), y además me devuelve el contacto actualizado. Y al final, `exportarJSON(contactos)` lo convierte a JSON bonito con `JSON.stringify(..., null, 2)`. También dejé una sección de pruebas con `console.log` donde agrego dos contactos, busco “mail”, actualizo uno, elimino otro y muestro el JSON final.

En **`App.jsx`** dejé la lógica principal de la Agenda ADSO. Ahí manejo el estado de los contactos con `useState` y hago que se guarden automáticamente en el navegador usando `localStorage`. Primero leo lo que ya esté guardado con `JSON.parse(localStorage.getItem('contactos'))` para que, si recargo la página, los contactos sigan ahí. Luego, con `useEffect`, cada vez que cambia el arreglo `contactos`, lo vuelvo a guardar con `localStorage.setItem('contactos', JSON.stringify(contactos))`. También tengo dos funciones clave: `agregarContacto`, que agrega un nuevo contacto al estado usando spread (`[...prev, nuevo]`), y `eliminarContacto`, que lo elimina filtrando por correo con `filter`. En la parte visual (JSX) personalicé el título con “Hola, soy Maicol - Agenda ADSO v1” y agregué un párrafo extra (`description`) contando lo que espero aprender en React. Finalmente renderizo el formulario con `<FormularioContacto onAgregar={agregarContacto} />` y muestro la lista usando `contactos.map(...)`, creando una `ContactoCard` por cada contacto.
